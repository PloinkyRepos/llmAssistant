llmAssistant Agent Documentation

This repository currently hosts one Ploinky agent: llmAssistant.

The agent provides LLM-backed helper tools consumed by Explorer and by other agents for code completion, commit-message generation, and conflict resolution.

All MCP tool invocations execute through llmAssistant/tools/llm_tool.sh and llm_tool.mjs.

What this agent does

  • Acts as intermediary between UI/agent requests and default Achilles LLM runtime calls.
  • Exposes a finite MCP contract from llmAssistant/mcp-config.json.
  • Normalizes MCP envelopes and validates tool inputs before LLM dispatch.
  • Provides reusable helper outputs for Git and editor workflows.
